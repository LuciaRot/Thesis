Introduction

The game design process can be summarized in 3 main parts:

  • Pre-production
  • Production
  • Post-production

    • This class mainly deals with the part of pre-production.
    At the end of this class you will be able to create a full game design document, which basically means you'll have a blueprint for your own game.

    Creating your own game helps improve your creativity and experience.

    All the literature mentioned in this course is either open source or accessable through the HFU library.

    This course was created as part of a bachelors thesis by Lucia Rothweiler at Hochschule Furtwangen University.
